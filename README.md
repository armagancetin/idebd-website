# IDEBD Website

Corporate website developed for IDE Bilişim Destek Hizmetleri.

## Overview

This project is a responsive corporate website designed to present company information, services, and contact details in a modern and user-friendly way.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Features

* Responsive design
* Modern user interface
* Company information pages
* Contact section
* Custom branding and visuals

## Project Structure

* `index.html` - Main page
* `style.css` - Styling and layout
* `script.js` - Interactive functionality
* `images/` - Website assets and logos
